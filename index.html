To use delegated auth to consume APIs just do something like: // Sample of an API consuming method const consumeAPI = token => { const xhr = new XMLHttpRequest(); xhr.open('GET', 'https://myawesomeapi.appspot.e3r.io'); xhr.setRequestHeader('x-jwt-auth', token); // ... xhr.send(); }; // And the actual way to consume the API is... AuthHelper.ensureAuth().then(onsumeAPI);
